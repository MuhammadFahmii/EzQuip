# EzQuip
EzQuip adalah sistem informasi pengelolaan barang berbasis PHP

# Dibuat dengan
1. PHP v7
2. Bootstrap 4
3. FontAwesome
4. JQuery

# Fitur
1. Terhubung API RajaOngkir
2. Penerapan AJAX pada beberapa halaman

# Penggunaan 
1. Clone repository ini
2. Import database ke MySql
3. Run Webserver
